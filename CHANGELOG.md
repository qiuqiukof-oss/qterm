# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/).

## [0.1.0] - 2026-07-20

### Changed
- 项目正式更名为 **Hesi（合思）**，原 CLI-Q 品牌退出；全仓库显示名称、包名（npm `hesi` / `hesi-tray`）、仓库链接同步更新。
- 版本号重置为 `0.1.0`，作为品牌重塑后的首个基线版本。

### Notes
- 运行期内部契约保留不变：`QCLI_*` 环境变量、`cli-q` MCP 服务器标识、`.cli-q-policy.json` 策略文件名均维持原样，以保证与现有部署 / 主机环境兼容。
- 代码仓库默认分支与远程地址（`qiuqiukof-oss/Hesi`）需由维护者同步重命名。
