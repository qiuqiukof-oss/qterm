Hesi — 在浏览器中运行任意 CLI

🚀 快速开始

1
选择 CLI 点击左侧 CLIs 列表中的任意工具
2
开始使用 终端自动启动，直接输入命令操作
3
随时切换 点击其他 CLI 即可切换，运行状态互不干扰

🤖 AI 智能体（一键安装）

点击 ⚡ 一键安装 即可集成智能体；U 盘版已随附离线包，插上即用、无需联网。开发版也可在左侧 Agent 面板 查看与管理。

🛠️ 其他Agent CLI（手动安装）

💻 Claude CodeAnthropic 终端代理
npm i -g @anthropic-ai/claude-code 需 Anthropic API Key
🧩 Codebuff自动化代码编写
npm install -g codebuff 文件操作和代码重构
🎯 mimocodemimocode 终端智能体
npm install -g mimocode 需 mimocode 账号 Token
🔮 CodexOpenAI Codex CLI
npm install -g @openai/codex 需 OpenAI API Key

⌨️ 快捷键

Ctrl+K打开命令面板
Ctrl+B切换侧边栏
Ctrl+Shift+F终端内搜索
Ctrl+IAI 对话
Ctrl+Shift+A全局搜索
Ctrl+=/-调整字体

💡 使用技巧

搜索框 过滤 CLI 列表 文件夹 分组管理工具 拖拽文件 上传 ⟳ 自动发现 CLI AI 助手感知终端上下文 URL/路径自动可点击

🚀 U 盘智能体 使用指南

⚡
插上即用 U 盘版集成 OpenCode 与 OhMyOpenAgent，无需联网安装
🤖
启动 OpenCode 终端输入 opencode 进入 AI 编程模式
🧩
多 Agent 编排 用 OhMyOpenAgent 组合多个智能体工作流
opencode "写个组件" 直接传参 --model claude-sonnet-4 指定模型 OMA 用 YAML/JSON 定义 Agent 流水线 文件操作 AI 读写 + diff 展示 Git 集成 建议 feature 分支使用 搭配 Hesi 管理多 Agent 会话

  • AI 生成代码务必 审查后再提交
  • 不要包含 API Key 等 敏感信息
  • 离线版模型需本地 / 私有部署支持
  • 大型重构分步骤进行
